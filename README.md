Measurement
===========

This project is a dashboard to show project status for agile team measurement.

####  How does it display like this?

Using front end data visualizaion framework [d3.js][src]

[src]: http://d3js.org/

####  Where those display data come from?

The data is synchronized with [Google spreadsheets][src] by using *sheetsee.js* and *tabletop.js* to take [spreadsheets as database][link]:

[src]: https://docs.google.com/spreadsheets/d/1zmjDANgPTlfbEygjCmESdkW1nsQCp257gkipq64sm3U/edit#gid=544972080
[link]: http://jlord.us/sheetsee.js/docs/basics.html
